O que são as Classes?

No CSS, classes são seletores usados para aplicar estilos a um ou mais elementos HTML. Elas permitem definir um conjunto de estilos que pode ser reutilizado em diferentes partes da página, ajudando a manter o código organizado e fácil de atualizar.

Principais Características das Classes

  1. Identificação por Ponto (.): No CSS, as classes são precedidas por um ponto (.). Por exemplo, se você definir uma classe .titulo, o CSS aplicará os estilos definidos a qualquer elemento com essa classe.
  2. Reutilização em Múltiplos Elementos: Diferente do id, que deve ser único por página, uma classe pode ser atribuída a diversos elementos, o que facilita a aplicação consistente de estilos a vários elementos ao mesmo tempo.
  3. Exemplo de Uso de Classe

    No HTML:

    No HTML:

    
<p class="destaque">
Este é um parágrafo destacado.
</p>

    No CSS:

    
.destaque {
    color: blue;
    font-weight: bold;
    background-color: lightyellow;
}
  4. Flexibilidade com Múltiplas Classes: Um elemento pode ter várias classes aplicadas a ele, permitindo combinações de estilos. Por exemplo, <p class="destaque importante"> aplicará os estilos de ambas as classes destaque e importante ao mesmo elemento.
  5. Especificidade e Prioridade: As classes têm uma especificidade intermediária no CSS, mais forte que um seletor de elemento, mas menos específica que um id. Isso significa que se o mesmo elemento tiver um estilo aplicado por um id e por uma class, o estilo do id terá prioridade.

Classes são fundamentais no CSS porque ajudam a criar layouts consistentes e reaproveitáveis, além de facilitar o gerenciamento de estilos em páginas complexas.
